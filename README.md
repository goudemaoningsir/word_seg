## 部署

### 1. 克隆项目

```bash
git clone https://github.com/sanmaomashi/word_seg_service.git
```

### 2. 安装依赖

```bash
pip install -r requirements -i https://mirror.baidu.com/pypi/simple
```

### 3. 执行项目

```bash
cd /项目目录/bin
bash start_project.sh
```
